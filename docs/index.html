Get started in seconds

Install the extension

Get it from the Edge Add-ons store and pin it to your toolbar.

Open the side panel

Click the extension icon in your Edge toolbar. The panel opens in the browser sidebar.

Export or Import

Click Export to XML to save your settings, or Import from XML to load a saved file.

Review the results

See a detailed summary of changed, unchanged, skipped, and failed settings. Restart Edge to apply all changes.
