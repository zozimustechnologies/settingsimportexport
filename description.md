# Edge Settings Export/Import

**Website:** [https://zozimustechnologies.github.io/settingsimportexport/](https://zozimustechnologies.github.io/settingsimportexport/)

Export and import your Microsoft Edge browser settings via XML files for easy transfer between devices.

## What it does

Transfer your complete Edge browser configuration — including content permissions, privacy preferences, and font settings — by exporting to a portable XML file and importing it on any device running Microsoft Edge.

## Key Features

- **Export to XML** — Save all your Edge settings to a portable file in one click
- **Import from XML** — Restore settings on any device from a saved file
- **Detailed Results** — See exactly what changed, what stayed the same, and what was skipped
- **Side Panel UI** — Clean, modern interface accessible from the Edge sidebar
- **Privacy First** — All data stays local; nothing is sent to external servers

## Settings Covered

- **Content Settings** — Cookies, images, JavaScript, notifications, popups, location, camera, microphone, auto-downloads
- **Privacy Settings** — Do Not Track, network prediction, safe browsing, search suggestions, referrer headers, spelling/translation services, autofill, password saving
- **Font Settings** — Default font size, fixed-width font size, minimum font size

## Privacy & Security

100% local processing. No servers, no accounts, no analytics. All data remains in your browser.
